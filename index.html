Veille Technologique

WEB 3.0

Qu'est ce que le WEB 3.0 ?

Le WEB 3.0 (ou web3) sera la nouvelle génération du WEB, elle succedera le WEB 1.0 et le WEB 2.0. Il sagit d'un web où tous les utilisateurs sont connéctés par des blockchains. Il se base sur un concept de liberté, ou tout les utilisateurs seraient affranchis des grandes entreprises.


Qu'est ce que la blockChain ?

La blockChain est une technologie de stockage et de transmition de l'information le tout sans organisme central de contrôle, cela permet la décentralisation. Pour faire simple, elle permet aux utilisateurs connectés de pouvoir partager des données sans intermediaire. On pourrait la comparer à une enorme BDD


Comment marche la blockChain ?

En pratique, la blockChain est une base de donnée qui contient l'historique de tous les échanges effectués entre les utilisateurs. Les principales caractéristiques pourrait être défini par :

- l'identification -> identidié chaque partie par un procédé cryptographique

- la transaction -> elle est envoyé à un réseau(ou noeud) d'ordinateur (=chaque noeud héberge une copie de la base de données dans lequel est inscrit l’historique des transactions effectuées.)

- la transaction -> elle est envoyé à un réseau(ou noeud) d'ordinateur (=chaque noeud héberge une copie de la base de données dans lequel est inscrit l’historique des transactions effectuées.)

- le système de sécurisation repose sur un mécanisme de consensus de tous les « nœuds » à chaque ajout d’informations. Les données sont déchiffrées et authentifiées par des « centres de données » ou « mineurs ». La transaction ainsi validée est ajoutée dans la base sous forme d’un bloc de données chiffrées (c’est le « block » dans blockchain)

- la décentralisation de la gestion de la sécurité empêche la falsification des transactions. Chaque nouveau bloc ajouté à la chaîne est lié au précédent et une copie est transmise à tous les « nœuds » du réseau. L’intégration est chronologique, indélébile et infalsifiable.
